如何使用西瓜静听播放 swf 文件

swf 扩展名的文件是 Flash 动画，曾经这种格式曾经带表了互联网富媒体应用的巅峰，然而如今却渐渐没落。Adobe 在 2020 年放弃了对 Flash 的技术支持，微软在 2021 年通过补丁移除了 Windows 自带的 Flash 控件。曾经能播放 swf 的软件（比如 PotPlayer）也放弃了对 Flash 的支持。

如果你希望能打开本地的 swf 格式的文件，可以使用西瓜静听，这是目前为数不多仍然支持播放 swf 的软件。

西瓜静听如何播放网络视频流

西瓜静听支持播放网络视频流，网络视频流指的是由网站或者程序发布的视频流的原始地址，比如：

  • m3u / m3u8 格式的 HTTP 地址
  • flv / mp4 格式的 HTTP / RTMP 地址
  • RTSP / RTP 地址

播放方式是在播放器中点击 “打开 URL”，在弹出的窗口中粘贴链接, 然后点击打开即可播放，由于网络视频位于远端服务器，所以播放效果受到你和服务器之间的网络状况影响较大。

URL： 统一资源定位符 俗称 网络地址
