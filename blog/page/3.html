西瓜静听命令行接口

可以通过命令行启动播放器,并通过参数对播放器进行初始状态的控制

PearPlayer.exe filepath options

options:
	-v number 音量(取值 0 ~ 500)
	-d number 播放速度(取值 100 ~ 100000, 1000 为常速, 2000 为 2 倍速)
	-o number 播放位置(秒数)
	-p number 是否置顶(0 / 1)
	-s number 是否精简模式(0 / 1)
	-l number 是否锁定列表(0 / 1)
	-f number 是否全屏(0 / 1)
	-m number 是否最大化(0 / 1)
	-w number 窗口宽度
	-h number 窗口高度
