软件使用过程中播放视频影片时闪退怎么办？

在主界面点击：右键-设置-解码，选择关闭硬件加速或内存零拷贝，尝试播放观察问题是否解决，依旧不行强烈建议您点击播放器右上角菜单按钮，然后点击关于项，通过上面的联系方式告知我们，我们将1对1地为您解决问题。

如何开启madVR渲染功能，提升视频画质

在程序初始安装阶段，勾选安装madVR，或者在主界面：右键-设置-渲染页面下载安装madVR。

安装成功后在右键-设置-渲染页面“madVR渲染”选项后勾选开，即可使用madVR进行渲染，同时也可通过madVR的控制栏面板控制视频的播放暂停等基本操作。

电脑以及西瓜静听使用时没有声音怎么办？

在主界面点击：右键-声音-声音设备，切换设备，查看问题是否解决。

通过西瓜静听显示Windows系统视频文件缩略图

视频文件缩略图是自从 Windows 7 及以后的版本包含的一个十分有用的特性，可以让我们在不打开视频文件的情况下了解视频大致内容，然而默认情况下 Windows 本身支持的视频格式少、效果不佳，所以出现了一些工具软件可对这一功能进行改善，比如：

  • Media Preview
  • Icaros
  • K-Lite Codec Pack

现在，我们凭借 西瓜静听 强大的解码能力，实现了对 Windows 视频文件缩略图功能的完善，为用户提供了一个新的选择。
