用西瓜静听播放网页视频

在播放器的社群中,我们常遇见这样的问题:

“为什么我在播放器中打开网页视频，无法播放？”

实际上播放器支持播放的视频流地址为在线的 mp4 或 m3u8 等流媒体地址，而不是播放页面的地址。

然而现在西瓜静听增加了对插件的支持，新版的播放器能通过插件对这种地址进行解析处理。

西瓜静听投屏

西瓜静听支持 DLNA 投屏协议，可以将手机 App 中的视频投到 PC 上的播放器中播放，也可以将 PC 上播放器中下载好的影片投到电视或者电视盒子上播放。

西瓜静听命令行接口

可以通过命令行启动播放器,并通过参数对播放器进行初始状态的控制

PearPlayer.exe filepath options

options:
	-v number 音量(取值 0 ~ 500)
	-d number 播放速度(取值 100 ~ 100000, 1000 为常速, 2000 为 2 倍速)
	-o number 播放位置(秒数)
	-p number 是否置顶(0 / 1)
	-s number 是否精简模式(0 / 1)
	-l number 是否锁定列表(0 / 1)
	-f number 是否全屏(0 / 1)
	-m number 是否最大化(0 / 1)
	-w number 窗口宽度
	-h number 窗口高度
